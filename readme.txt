This plugin allows you to quickly and easily integrate your wordpress website to Convizit analytics system by simply writing your Token ID.


== Description ==

Convizit analytics allows you to see different metrics regarding your visitors behavior. Convizit specializes in providing
deep level analytics absolutely automatically after 30 seconds installation. 
Integrate your wordpress website quickly and easily using this plugin.


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go To Convizit.com and login to your account
4. Click the </> (tracking code) button in the upper menu
5. Copy your Token ID
6. Go to wodpress admin panel, click Settings->Convizit Analytics and paste the token id.
7. Click save changes
8. You're Done! :)


== Frequently Asked Questions ==

= Where can I get a token ID? =

After you'll sign up at https://convizit.com/register.php, login to your account and click the </> (tracking code) button in the upper menu.
You'll find your Token ID right there. 

= Is there anything else I should do? =

After placing your Token ID, Convizit will automatically track your visitors behavior.

= Where can I see my website metrics? =

You can find your deep level analytics metrics in your account at https://convizit.com/welcome.php
